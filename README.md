# Point Cloud Rendering

一个基于 Web 的点云可视化和标注工具，支持 PCD 格式文件的加载、交互式选择和着色标注。

## 技术栈

- **框架**: Next.js 16.0.7
- **UI**: React 19.2.0
- **3D 渲染**: Three.js + React Three Fiber
- **样式**: Tailwind CSS 4.1.9
- **语言**: TypeScript 5
- **包管理器**: pnpm

## 环境要求

- Node.js 18.x 或更高版本
- pnpm 8.x 或更高版本

## 快速开始

### 1. 安装依赖

```bash
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

服务器将在 [http://localhost:3000](http://localhost:3000) 启动
